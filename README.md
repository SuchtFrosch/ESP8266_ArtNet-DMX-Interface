## ESP8266_ArtNet-DMX-Interface
Receiv with a ESP8266 ArtNet and send DMX.

This project Receiv with a ESP8266 ArtNet Packages and send DMX Signals.

# Material
ESP8266 D1 Mini
SN751768P
DMX-Socket
RGB-LED

# Shematik
RGB-LED:
        R - ESP8266 D2
        G - ESP8266 D1
        B - ESP8266 D0
SN751768P:
        1 - ESP8266 D4
        2 - ESP8266 GND
        3 - ESP8266 VCC 5V
        4 -
        5 - ESP8266 GND
        6 - DMX-Socket 3
        7 - DMX-Socket 2
        8 - ESP8266 VCC 5V

# Powersource over the micro-USB port of the ESP8266.
